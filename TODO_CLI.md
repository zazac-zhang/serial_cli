## 🔧 待优化功能

### 1. DTR/RTS 平台特定实现（中优先级）
**位置**: `src/serial_core/port.rs` - `platform_signals` 模块

**当前状态**: 框架已完成，日志记录正常，但底层信号控制未激活

**优化内容**:
- Unix: 使用 `ioctl` + `TIOCMSET` 标志控制
- Windows: 使用 `EscapeCommFunction` API

**影响**: 不影响日常使用，仅对需要精确控制 DTR/RTS 时序的工业设备有必要

**影响**: 当前实现已满足大多数场景，高级并发场景可优化
